# angular-mongo-express-three-node

```
run npm install 
```

## Start MongoDB:

### for Ubuntu
```
sudo systemctl start mongod
``` 

#Verify that MongoDB has started successfully:
for Ubuntu
```
sudo systemctl status mongod
```

#Start node js server 

in /angular-three-node folder 
```
npm run start 
```

in /test-app folder
```
ng serve
```
